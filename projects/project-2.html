Receptenwebsite

Een website waar gebruikers recepten kunnen opzoeken en bewaren.

Over het project

Voor dit project moest ik een website maken waar mensen recepten kunnen vinden. De opdracht was om het simpel te houden maar wel goed werkend.

Ik heb gekozen voor een kaartensysteem waar elk recept zijn eigen card heeft met een foto. Zo zie je meteen wat het is.

Hoe ik het heb aangepakt

Eerst heb ik een schets gemaakt op papier. Daarna ben ik beginnen coden met HTML en Bootstrap. Voor de kleuren heb ik gekeken naar populaire receptsites en wat daarvan overgenomen. 


// Filter functie voor recepten
function filterRecipes(category) {
  var cards = document.querySelectorAll('.recipe-card');
  for (var i = 0; i < cards.length; i++) {
    cards[i].style.display = 'block';
  }
}

Resultaat

De website werkt goed en ziet er netjes uit. Je kan makkelijk door de recepten scrollen en alles is responsive op je gsm.

Wat ik heb geleerd

Ik heb geleerd hoe je cards maakt in Bootstrap en hoe je foto's goed laat passen zonder dat ze vervormd zijn. Ook heb ik meer ervaring met CSS Grid gekregen.
