Weer App

Een simpele applicatie om het weer op te zoeken per stad.

Over het project

Dit was mijn eerste project met een API. Ik wou leren hoe je data van internet kan halen en op je website kan zetten. Een weer app leek me een goed begin.

Je kan een stad ingeven en dan zie je de temperatuur, of het regent en de wind.

Hoe ik het heb gemaakt

Ik heb een gratis weer API gevonden online. Dan heb ik geleerd hoe fetch() werkt om data op te halen. Het was lastig maar uiteindelijk kreeg ik het werkend. 


// Weer ophalen
function getWeather(city) {
  fetch('https://api.weather.com/data?city=' + city)
    .then(response => response.json())
    .then(data => {
      document.getElementById('temp').textContent = data.temp;
    });
}

Resultaat

De app werkt! Je kan elke stad opzoeken en je ziet meteen het weer. Ik heb ook icoontjes toegevoegd voor zon, regen en wolken.

Wat ik heb geleerd

Dit was best moeilijk maar ik heb veel geleerd over API's en asynchrone JavaScript. Vooral het debuggen van fetch errors was een goede oefening.
